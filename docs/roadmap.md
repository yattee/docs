# Roadmap

This page outlines planned features and improvements for the Yattee ecosystem. Items are listed in no particular order and are subject to change.

Yattee is developed mostly by a single person, so your patience is greatly appreciated as these features are worked on. Thank you for your feedback and support!

## Client App

### macOS Release
Bringing Yattee 2 to macOS beyond the current iOS/iPadOS and tvOS support. tvOS is now available via TestFlight; macOS work is in progress.

### Improved Accounts Management
Allow using playlists, subscriptions, and other account data from multiple source types simultaneously, making it easier to aggregate content across different backends.

### Video Player Customization
More layout options and customization for the player screen, including sidebar layout, configurable video info display, and customizable context menu actions to better suit different viewing preferences and workflows.

### Audio-Only Mode
Option to disable video playback for audio-only listening.

### Queue Management
More options for managing how the videos are managed in the queue.

### Deep Linking Fixes
Fixes and improvements for handling external links (e.g. opening YouTube URLs from description), ensuring a smoother experience when sharing and opening links.

### Bug Fixes
Ongoing work on fixing reported bugs and improving overall stability of the app.

## Yattee Server

### Improved Channel Refresh
Better channel refresh mechanism, removing the current 500-channel limit and allowing larger channel libraries to stay up to date.

### User Feed & Channel Pages
Dedicated pages for user feed and channels with inline video playback, providing a more complete browsing experience directly in the web interface.

### iCloud replacement
Yattee Server should allow to store and sync data between devices that are currently managed via iCloud only.
